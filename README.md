bison
=====

Bison 是一个JAVA 程间的通信框架，基于apache mina 实现，对mina进行了byteBuffer 缓冲区重用以及半包出处时减少拷贝。

客户端(bison-client) 功能点

1 业务分组。

2 支持分组内负载均衡。

3 支持横向扩展。

服务端(bison)

1 支持热部署

2 支持流控，防止服务端出现拥堵时，客户端请求可以快速失败。

3 资源统一管理。数据库连接池，工作线程,等。

4 支持白名单，黑名单。

计划

1 支持netty做底层通信。
